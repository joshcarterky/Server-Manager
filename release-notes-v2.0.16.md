# Dedicated Server Manager v2.0.16 Release Notes

This release fixes several server configuration and console workflow issues.

## Changes

- Improved the Logs tab with parsed entries, counts, details, and copy/export controls.
- Added the missing BattleEye setting to the INI editor.
- Fixed the RCON console so the command bar stays visible when output grows.
- Fixed Max Players and other managed settings so startup uses the current saved value instead of stale template data.
