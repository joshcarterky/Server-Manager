# Dedicated Server Manager v2.0.20 Release Notes

This release cleans up the RCON console output.

## Changes

- Raw ASA server stdout and stderr logs no longer flood the RCON console.
- Server process output is still written to the application logs for troubleshooting.
- RCON console output is kept focused on connection status, commands, and command responses.
