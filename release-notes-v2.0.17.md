# Dedicated Server Manager v2.0.17 Release Notes

This release fixes INI-backed server settings loading stale values after selecting a server.

## Changes

- Selected servers now reload matching editor values from `Game.ini` and `GameUserSettings.ini`.
- Configure Server and the INI Settings Editor now start from the current server INI files when those files exist.
