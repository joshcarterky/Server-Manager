# Dedicated Server Manager v2.0.11 Release Notes

**Release date:** 2026-06-03

## Overview
This release adds in-app application updating.

## What changed
- Added a Download & Install button in Settings after update checks.
- The app downloads the update zip, stages it, closes, replaces the application files, and relaunches.
- Open Download remains available as a manual fallback.
