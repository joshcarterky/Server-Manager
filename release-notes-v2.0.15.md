# Dedicated Server Manager v2.0.15 Release Notes

**Release date:** 2026-06-03

## Overview
This release adds the new Server Manager icon and improves Add Existing Server imports.

## What changed
- Added the new icon to the EXE, main window, taskbar, and tray icon.
- Add Existing Server now copies the selected existing install into the app's default `servers` folder.
- Imported servers now point at the copied default-location folder instead of the original source folder.

- Asset: `release/Dedicated-Server-Manager-v2.0.15-win-x64.zip`
