# Dedicated Server Manager v2.0.10 Release Notes

**Release date:** 2026-06-03

## Overview
This release improves the RCON console controls.

## What changed
- Added a Refresh button to reconnect the selected server's RCON console.
- Kept Clear available in the RCON console control row for quickly clearing output.
- Matched the controls across the server tab and dashboard RCON console layouts.
