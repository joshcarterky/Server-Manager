# Dedicated Server Manager v2.0.7 Release Notes

**Release date:** 2026-06-03

## Overview
This release fixes the INI Settings Editor workflow and ships the latest Windows x64 update package.

## What changed
- Fixed the INI Settings Editor crash caused by missing WPF resources.
- Restyled the INI Settings Editor to match the app's dark theme.
- Added an explicit Save button to the INI Settings Editor.
- Fixed INI editor saves so changes refresh Configure Server settings and persist to INI files.
- Fixed Configure Server name edits being overwritten by stale INI values.

- Asset: `release/Dedicated-Server-Manager-v2.0.7-win-x64.zip`
- Version: `2.0.7`
