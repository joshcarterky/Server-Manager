# Dedicated Server Manager v2.0.12 Release Notes

**Release date:** 2026-06-03

## Overview
This release improves RCON console access and cleans up the Templates tab for multiple games.

## What changed
- Added visible Reconnect and Clear buttons to the Configure Server RCON console header.
- Renamed the RCON refresh action to Reconnect for clearer wording.
- Organized templates by game with section headers and template counts.
- Simplified template cards by removing repeated game labels inside each card.
